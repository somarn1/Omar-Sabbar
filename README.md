# project final 

<!--  Delicious Restaurant Website -->

A modern and fully responsive restaurant website built using HTML, SCSS, and Bootstrap 5.

<!--  Overview -->

This project is a complete restaurant website that includes:

Fixed top navigation bar

Hero image carousel

About section

“Why Us” features

Specials with tabs

Events section

Table reservation form

Gallery

Chefs section

Contact form with Google Maps integration

Footer

The entire layout is responsive thanks to Bootstrap and custom SCSS styling.
